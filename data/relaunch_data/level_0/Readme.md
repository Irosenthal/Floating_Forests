# Level 0 Data Products for Floating Forests

These products are not complete datasets, and might not be useful for public consumption

- ff_relaunch_subjects.rds - metadata of uploaded subjects in Rds format             
\
- floating_forests_classifications.rds - raw output of classification workflow in Rds format  
\
- floating_forests_swipe.rds - raw output of swipe kelp/no kelp interface             
\
- utm_zone_XX_sf.Rds - R data file for UTM zone XX consisting of sf object of raw user classifications
\
\

## Intermediate Tiles for Consensus processing
raster_tiles - consensus rasters of individual subjects  
\
sf_tiles - polygon tiles of individual subjects derived from rasters
